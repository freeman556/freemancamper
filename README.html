🎯 这个网站是什么？

这是一个皮卡露营装备展示网站，包含：

  • 🏠首页：大图展示、产品分类、精选产品
  • 📦产品中心：产品列表、产品详情
  • 🔧配置器：可以选配不同配置，自动计算价格
  • 📸客户案例：展示用户使用案例
  • ❓FAQ：常见问题解答
  • 📝在线询价：客户可以填写询价表单
  • 🔐后台管理系统：可以修改网站内容

⚠️ 重要注意事项（必看！）

1. 关于后台修改的数据

目前的后台管理系统，修改的数据只存在你自己的浏览器里！

什么意思呢？

  • ✅ 你在自己电脑上修改后，你自己能看到效果
  • ❌ 其他访客访问你的网站，看不到你修改的内容
  • ❌ 换个浏览器、换台电脑、清除浏览器缓存，修改的内容就没了

为什么会这样？

因为这是一个纯静态网站，没有后端服务器。数据存在浏览器的「本地存储」里，就像存在你自己电脑的记事本里，别人看不到。

那怎么让所有人都看到修改后的内容？

有两种方法：

  • 方法一（推荐）：在后台改好后，用「数据导出」功能导出 JSON 文件，然后替换到代码里
  • 方法二：直接修改代码里的默认内容，然后重新上传到 GitHub

2. 关于图片

网站里的产品图片、首页大图等，建议用网络图片链接（URL），不要用本地图片。

  • 可以把图片上传到图床（比如微博图床、阿里云OSS、七牛云等）
  • 然后把图片链接填到后台里

🚀 部署步骤：上传到 GitHub + 连接 Netlify

第一步：注册账号

你需要注册两个账号（都是免费的）：

  1. GitHub：https://github.com （用来存网站代码）
  2. Netlify：https://www.netlify.com （用来把网站放到网上）

第二步：把网站上传到 GitHub

方法一：网页上传（最简单，推荐小白）

  1. 登录 GitHub，点击右上角的 + 号，选择 New repository
  2. 填写仓库信息：
    • Repository name：随便起个名字，比如 freeman-camper
    • Public：选这个（公开的，Netlify 才能访问）
    • 不用勾选其他选项
  3. 点击 Create repository
  4. 找到页面中间的 uploading an existing file 链接，点它
  5. 打开你电脑上的 freeman-camper 文件夹，把里面所有文件和文件夹都拖进去
    注意：是拖文件夹里面的内容，不是拖整个文件夹！
  6. 拖完后，点击页面底部的 Commit changes
  7. 完成！代码已经上传到 GitHub 了

第三步：连接 Netlify，把网站放到网上

  1. 登录 Netlify：https://app.netlify.com
  2. 点击 Add new site → Import an existing project
  3. 选择 GitHub（第一次用会让你授权，点同意就行）
  4. 找到你刚才创建的仓库，点它
  5. 部署设置页面：
    • Branch to deploy：选 main 或者 master（一般默认就行）
    • Build command：留空！什么都不用填
    • Publish directory：留空！或者填 .（就是当前目录）
  6. 点击 Deploy site
  7. 等待 1-2 分钟，网站就部署好了！

第四步：查看你的网站

部署成功后，Netlify 会给你一个网址，类似：

https://随机名字.netlify.app

点击这个网址，就能看到你的网站了！

🎉 恭喜！你的网站已经上线了！

第五步（可选）：绑定自己的域名

如果你有自己的域名，可以在 Netlify 后台绑定：

  1. 进入网站设置，找到 Domain settings
  2. 点击 Add custom domain
  3. 按照提示操作，需要去域名服务商那里改 DNS 解析
  4. 不会弄的话，可以搜一下「Netlify 绑定域名」教程，很多的

🔐 后台管理系统怎么用？

怎么打开后台？

在你的网站地址后面加上 /admin/ 就行。

比如：

  • 你的网站是 https://xxx.netlify.app
  • 后台地址就是 https://xxx.netlify.app/admin/

默认密码：admin321

后台能改什么？

登录后台后，左边有菜单，可以修改：

  1. 📦 产品管理
    • 添加新产品
    • 编辑已有产品（名字、价格、图片、描述等）
    • 删除产品
  2. 📂 分类管理
    • 添加产品分类
    • 编辑分类名称和图标
    • 删除分类
  3. 📝 表单管理
    • 查看客户提交的询价表单
    • 删除表单记录
  4. ⚙️ 网站配置
    • 修改网站名称
    • 修改联系方式（电话、邮箱、地址）
    • 上传/修改 Logo
    • 上传/修改首页大图
    • 修改首页的标题和副标题

✏️ 怎么修改网站默认内容（让所有人都能看到）

如果你想让所有访客都看到修改后的内容，需要把后台修改的数据导出，然后替换到代码里。

最简单的方法：后台导出 + 替换文件

  1. 登录后台，修改好产品、分类、配置等所有内容
  2. 进入「全局设置」→「数据管理」
  3. 点击「📤 导出数据」，会下载一个 freeman-data.json 文件
  4. 打开下载的 JSON 文件，全选复制
  5. 打开 assets/js/data.js 文件，找到 var defaultData = { 这一行
  6. 把后面的内容替换成你复制的 JSON 数据
  7. 保存文件，重新上传到 GitHub
  8. 等 1-2 分钟，Netlify 会自动更新网站！

方法二：直接在 GitHub 上编辑

如果你只是改一两个地方，也可以直接在 GitHub 上编辑：

  1. 打开你的 GitHub 仓库
  2. 找到要修改的文件，点击进去
  3. 点击右上角的铅笔图标（Edit this file）
  4. 修改内容
  5. 拉到页面底部，点击 Commit changes
  6. 等 1-2 分钟，Netlify 会自动更新网站！

具体改哪些文件？

1. 修改产品、分类、案例、FAQ 等内容

主要改这个文件：assets/js/data.js

用文本编辑器打开，找到对应的部分修改就行。比如：

  • categories 是分类
  • products 是产品
  • cases 是案例
  • faqs 是常见问题

2. 修改配置器的产品和价格

配置器的数据在 configurator.html 文件里，是内嵌的 JS 代码。

这个比较复杂，建议找懂点代码的人帮你改，或者用后台的配置器管理功能改好后导出。

3. 修改首页文字、联系方式等

也是在 assets/js/data.js 里的 config 部分。

❓ 常见问题

Q1：我改了后台，为什么别人看不到？

A：因为数据存在你自己的浏览器里。想让所有人看到，需要导出数据替换到代码里，参考上面「怎么修改网站默认内容」。

Q2：询价表单提交后，我怎么收到通知？

A：我已经帮你配置好了 Netlify Forms！部署后：

  1. 客户填写询价表单并提交
  2. 你可以在 Netlify 后台的 Forms 菜单里看到所有提交
  3. 还可以设置邮件通知，有人提交就发邮件告诉你

设置邮件通知的方法：

  1. 进入 Netlify 网站后台
  2. 点击 Forms 菜单
  3. 找到 Notifications
  4. 添加邮件通知，填你的邮箱就行

另外，表单数据也会存在浏览器本地，你可以在网站后台的「表单管理」里看到（但只有你自己的浏览器能看到）。

Q3：配置器的询价我也能收到吗？

A：可以！我也帮你配置好了。配置器的「提交配置获取报价」功能也接入了 Netlify Forms。

  • 客户选好配置后填写联系信息提交
  • 你可以在 Netlify 后台的 Forms 菜单里看到，表单名叫 configurator-quote
  • 里面包含客户选的所有配置详情和联系信息
  • 同样可以设置邮件通知

Q4：图片上传不了怎么办？

A：纯静态网站不能直接上传图片。建议：

  1. 把图片上传到图床（比如阿里云OSS、七牛云、微博图床等）
  2. 复制图片链接
  3. 粘贴到后台的图片地址输入框里

Q5：怎么修改后台密码？

A：有两种方法：

  1. 方法一（推荐）：登录后台，进入「全局设置」→「修改管理员密码」，直接改就行（但只存在当前浏览器）
  2. 方法二：直接修改代码里的默认密码，在 admin/index.html 里搜索 admin321 改成你想要的密码

Q6：网站打开慢怎么办？

A：可能是图片太大了。建议：

  • 图片压缩后再上传
  • 用 CDN 加速图片
  • 产品图片不要太大，控制在 500KB 以内比较好
